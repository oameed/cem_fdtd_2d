# FDTD for 2D TEz Problems

This project demonstrates application of the Finite Difference Time Domain (FDTD) method to the following problems:
* 2D cavities  : rectangular and circular 
* 2D scattering: PEC/Dielectric rectangular and circular objects

## References

[1]. 2011. Inan. _Numerical Electromagnetics: The FDTD Method_  
[2]. 2009. Elsherbeni. Demir. _The FDTD Method for Electromagnetics_ 2nd Edition  
[3]. 2007. Berenger. _Perfectly Matched Layer (PML) for Computational Electromagnetics_  
[4]. 2005. Taflove. Hagness. _Computational Electrodynamics_ 3rd Edition  

## How to Run

* To run simulations (with problem 1 as an example):
  1. `cd` to the main project directory
  2. `./run/run_problem_1.sh`
